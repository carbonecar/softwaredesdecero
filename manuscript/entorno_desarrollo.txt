#Generando el entorno de desarrollo
Para desarrollar, lo vamos a hacer en java.
Entonces vamos a definir:
a)Version de java
b)Herramienta para la construccion y gestion del softwar y componente
c)Repositorio de componentes
e)Repositorio de fuentes
f)Entornos de despligue (desarrollo, testing y produccion)
g)Herramientas para la automatizacion del proceso de construccion y testeo
